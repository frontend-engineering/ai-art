# 背景音乐资源

请将背景音乐文件放置在此目录下。

## 文件要求

- **文件名**: `bgm.mp3`
- **格式**: MP3
- **建议时长**: 2-5 分钟（循环播放）
- **建议大小**: < 2MB（小程序包大小限制）
- **建议码率**: 128kbps

## 使用说明

1. 准备一个适合春节喜庆氛围的背景音乐
2. 将音乐文件重命名为 `bgm.mp3`
3. 放置到此目录 (`miniprogram/assets/audio/`)
4. 音乐将在用户点击音乐控制按钮后播放

## 注意事项

- 确保音乐版权合规
- 小程序需要用户交互后才能播放音频
- 音乐会在小程序切换到后台时自动暂停
