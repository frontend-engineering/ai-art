一、交互逻辑总纲

核心原则：全链路操作≤5步，老年友好（大按钮、语音引导），春节氛围贯穿，每步操作均有明确反馈，无断点闭环。

二、核心交互流程（用户旅程+系统反馈）

1. 启动-登录环节

- 用户操作：打开小程序

- 系统反馈：展示3秒启动页（自动轮播情感图），3秒后自动跳转；支持右上角“跳过”

- 交互细节：跳转后直接触发微信快捷登录，1秒完成，自动获取微信昵称，无额外注册步骤

2. 功能选择环节

- 用户操作：点击“时空拼图”或“富贵变身”卡片

- 系统反馈：点击卡片时有0.1s缩放+红色边框高亮反馈，跳转对应上传页

- 交互细节：底部“查看30秒教程”文字链，点击弹出短视频教程（自动播放，支持关闭）

3. 图片上传环节

3.1 时空拼图上传

- 用户操作：点击圆形上传框→选择相册多选/拍照

- 系统反馈：


  - 上传成功：2秒内显示“✓ 人脸检测成功”，上传框显示缩略图，右上角带“×”删除按钮

  - 上传失败：显示“⚠️ 人脸模糊/未检测到人脸”，配示例图参考，引导重拍

- 交互细节：支持语音指令“添加爸爸的照片”，自动打开相册；上传满2人后，底部“下一步”按钮点亮（默认置灰）

3.2 富贵变身上传

- 用户操作：点击大尺寸上传框→上传现有全家福

- 系统反馈：支持拖拽上传，上传中显示进度条，完成后显示“✓ 上传成功”

- 交互细节：提示“支持JPG/PNG，最大10MB”，自动检测图片格式，不合格则弹窗提示

4. 模板选择环节

- 用户操作：横向滚动选择模板，点击选中

- 系统反馈：选中模板后金色边框高亮+“✓”标记，点击底部“生成”按钮跳转等待页

- 交互细节：默认选中“新中式团圆”模板，减少决策成本；滚动时模板卡片有顺滑过渡效果

5. 生成等待环节

- 用户操作：等待生成

- 系统反馈：
        

  - 展示旋转灯笼动画+动态进度条，同步显示文案：20%识别人脸→40%调和光线→60%融合背景→80%优化细节→100%完成

  - 显示“您前面还有X位用户，预计等待X秒”，底部“先看别人的作品”按钮（点击跳案例页）

- 交互细节：生成成功后自动跳转结果筛选页；失败则弹窗提示“生成失败，点击重试”，支持一键重试

6. 结果筛选环节

- 用户操作：点击查看4宫格结果，选择1张满意的→点击底部“确认选择”

- 系统反馈：
        

  - 图片加载：采用“掀布”动效（从模糊到清晰），增强惊喜感

  - 选中反馈：点击图片放大预览（支持双指缩放），选中后红色边框+“✓ 选中”标记

- 交互细节：仅选中1张后，“确认选择”按钮点亮，点击后跳转成果页

7. 成果页-增值操作环节

- 用户操作：点击“下载高清图”“生成贺卡”“定制晶瓷画”“分享”按钮

- 系统反馈：
        

  - 下载高清图：弹出付费弹窗，选中套餐时有金色边框反馈，点击支付触发微信支付

  - 生成贺卡：跳转贺卡编辑页，支持自定义祝福语（实时预览效果）

  - 定制晶瓷画：跳转规格选择页，选择后显示产品预览图

  - 分享：自动生成带小程序码的分享图+春节文案，直接跳转微信分享界面

- 交互细节：成果页高清图自动播放5秒微动态，长按可保存带水印预览图（带“保存成功”提示）

三、UI/UED核心规范

1. 基础视觉规范

- 色调：主色中国红（#D4302B），辅色金色（#D4AF37），背景暖白（#FFF8F0）

- 字体：
        

  - 标题：24-28px，汉仪尚巍手书（节日感）

  - 正文：18px，苹方/思源黑体（清晰易读）

  - 说明文字：16px，浅灰色

  - 老年模式：所有字体放大20%，按钮尺寸放大15%

- 圆角与阴影：所有按钮/卡片8px圆角，卡片hover时轻微阴影（增强层次感）

- 节日元素：全程穿插灯笼、烟花、福字、红绸元素，不堆砌（如按钮边缘红绸装饰、等待页旋转灯笼）

2. 交互组件规范

2.1 按钮设计

- 尺寸：所有可点击按钮≥48×48px（老年友好）

- 反馈：点击时有0.1s缩放（缩小10%）+ 背景色加深反馈

- 样式：
        

  - 主要按钮（立即制作、确认选择）：红色渐变背景，白色粗体字

  - 次要按钮（教程、重试）：白色背景，红色边框+文字

  - 功能按钮（下载、分享）：对应功能色（下载金色、分享灰色），带直观图标

2.2 弹窗设计

- 样式：圆角12px，白色背景，红色边框顶部装饰

- 交互：弹出/关闭时有0.3s淡入淡出效果，支持点击外部关闭（除付费弹窗）

- 付费弹窗特殊：不可点击外部关闭，底部有“微信支付安全加密”信任标识

2.3 加载与提示设计

- 加载动画：统一使用旋转灯笼动画（80×80px），避免单调转圈

- 提示文案：
        

  - 成功提示：绿色“✓”+ 黑色文字（如“保存成功”）

  - 错误提示：红色“⚠️”+ 红色文字（如“人脸模糊，建议重拍”）

  - 引导提示：蓝色“i”+ 黑色文字（如“请上传清晰正面照”）

3. 关键页面UI细节

3.1 启动页

- 布局：全屏轮播3张情感图（留守儿童盼父母、异地视频通话、AI全家福成品）

- 文案：居中展示主标题“这个春节，让爱没有距离”，副标题“3步生成惊艳全家的AI全家福”，底部动态数字“已帮助12,345个家庭团圆”

- 按钮：底部红色“立即制作”按钮（48×160px），右上角“跳过”文字按钮

3.2 功能选择页

- 布局：上下两个大卡片（各占50%屏幕高度），居中展示

- 卡片A（时空拼图）：图标🧩，标题“家人天各一方？”，副标题“拼出大团圆”，示例图展示“3张单人照→合成故宫全家福”

- 卡片B（富贵变身）：图标👑，标题“背景太土？”，副标题“一秒变豪门”，示例图展示“杂乱餐桌→欧式豪宅背景”

- 底部：“有疑问？查看30秒教程”灰色文字链

3.3 上传照片页

- 顶部：大字语音引导文案“请上传清晰正面照，光线越亮效果越好”（18px，红色）

- 上传区域：
        

  - 时空拼图：2-5个圆形上传框（直径80px），带“+添加家人”占位图，上传后显示缩略图

  - 富贵变身：1个大尺寸上传框（280×200px），白色背景，红色边框，带“点击上传全家福”占位文字

- 底部：蓝色渐变“下一步”按钮（48×260px），未满足条件时置灰

3.4 成果页

- 展示区：顶部60%屏幕展示高清图，左上角“Live”标识（微动态标识），图片带红色印章水印“AI团圆照相馆”（带小程序码，不遮挡人脸）

- 功能按钮区：垂直排列4个功能按钮（各48×48px，带图标+文字），从左到右依次：

  - 金色“下载高清图”

  - 红色“生成拜年贺卡”

  - 紫色“定制晶瓷画”

  - 灰色“分享家族群”

3.5 付费弹窗

- 布局：纵向排列3个套餐卡片，带明确权益说明，顶部标题“选择套餐解锁高清全家福”

- 套餐卡片：
        

  - 免费版：0元，白色背景，权益“标清带水印，限2人合成”

  - 9.9元尝鲜包：红色背景，价格划掉原价19.9元，权益“高清无水印+3-5人合成+热门模板”

  - 29.9元尊享包：金色背景+“85%用户选择”标签，权益“4K原图+微动态+贺卡+全模板+优先队列”

- 底部：“微信支付安全加密”“支付后永久保存”灰色小字

4. 老年友好专项UI设计

- 大字模式开关：首页右上角设置“老年模式”按钮（带老人图标），开启后字体+20%、按钮+15%

- 语音引导：关键页面（上传、生成）自动播放语音提示，支持“关闭语音”按钮

- 简化界面：老年模式下隐藏非核心元素（如“看别人作品”“教程”），仅保留核心操作按钮

四、体验优化关键细节

- 惊喜感设计：生成成功后，屏幕绽放1秒烟花动画，配“团圆成功”音效

- 节气文案：根据日期自动切换页面顶部文案（如除夕“除夕团圆，万事顺遂”）

- 错误提示通俗化：不说“参数错误”，说“照片里人脸太小啦，选一张正面大头像吧”

- 背景音乐：所有页面支持开启春节轻音乐（默认关闭），底部带“音乐”开关图标
