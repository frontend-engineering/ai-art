# AI全家福管理后台

> 功能完整的管理后台系统，用于管理AI全家福小程序的运营数据

## 🚀 快速开始

```bash
# 1. 启动后端服务
cd backend && pnpm run dev

# 2. 启动管理后台前端（新终端）
cd admin && pnpm run dev

# 3. 访问管理后台
# URL: http://localhost:5173
# 默认账号: admin
# 默认密码: Admin@123456
```

## ✨ 已实现功能

- 🔐 **认证系统** - JWT认证、角色权限、会话管理
- 💰 **价格管理** - 动态配置、历史记录、API集成
- 👥 **用户管理** - 列表查看、详情展示、状态管理、数据导出
- 📦 **订单管理** - 统一视图、状态更新、退款处理、数据导出
- 📊 **数据看板** - 实时统计、趋势分析、图表展示

## 🛠️ 技术栈

**前端**: React 18 + TypeScript + Vite + Ant Design 5 + ECharts

**后端**: Node.js + Express + MySQL + JWT

**工具**: pnpm

## 🔄 价格配置迁移

价格配置已从硬编码迁移到数据库，支持动态管理：

1. **后端**: 已集成价格API，带降级方案
2. **小程序**: 需配置 `apiBaseUrl` 并调用价格API
3. **H5**: 已集成 `usePrices` Hook
4. **云函数**: 已集成价格API，需配置环境变量 `API_BASE_URL`

所有端都实现了5分钟缓存和降级机制，确保支付流程稳定。
